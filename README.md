# Myapp

This project is front-end design and functionality for the glossary moduels. Users can perform Create, Read, Update and Delete

### Setup the Application
####  Clone the project

$ clone this repo

$ cd {root-directory}

$ npm install

$ change API endpoint url in glossary.service.ts

$ npm start
